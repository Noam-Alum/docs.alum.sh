## gen_random

Generate random data.

Options:
- int | Only use numbers.
- str | Only use alphabetic characters.
- all | Use anything.

**For example:**
```bash
gen_random int 14
```
**Would return:**<br>
36261766974132
 
::: tip uc variables
`uc_gr_len` - Change default length (*12*).<br>
For example:
```bash
uc_gr_len=8
```
:::
